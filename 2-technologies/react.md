
**React** is a JavaScript library with tools to build ==UI components== (functions that take data and <span style="color:#ffff00">return</span> HTML-like markup) and then use those <span style="color:#92d050">components</span> as<span style="color:#4983d0"> building blocks </span>for complex, data-driven, dynamic interfaces.

---

## reverse flow

instead of starting with HTML/CSS/JS markup , React starts with JS and returns markup based on states and user's actions.
